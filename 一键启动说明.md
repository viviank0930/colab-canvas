# CoLab 一键启动

## 第一次使用

1. 双击 `首次设置.command`。
2. 粘贴 DeepSeek API Key。输入不会显示在屏幕上，这是正常的。
3. 设置完成后关闭窗口。

API Key 保存在 macOS 钥匙串中，不会写入 HTML、JavaScript、日志或项目文件。

## 以后使用

- 双击 `启动 CoLab.command`：后台启动服务器并自动打开 Chrome 默认浏览器。
- 双击 `停止 CoLab.command`：关闭本地服务器。

网页和交互代码可以继续修改。只要保留 `server.py` 和前端的 `/api/analyse` 请求，DeepSeek 接入就会一直存在。

如需更换 API Key，再次运行 `首次设置.command` 即可覆盖钥匙串中的旧 Key。
